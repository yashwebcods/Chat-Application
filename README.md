# 💬 ChatApp – Real-Time MERN Stack Chat Application

Welcome to **ChatApp**, a modern real-time chat application built with the **MERN stack**. It offers seamless messaging functionality with a stylish UI powered by **Tailwind CSS** and **DaisyUI**, and real-time updates using **Socket.IO**.

## 🚀 Tech Stack

### Frontend
- ⚛️ **React 19**
- 🛠️ **Zustand** – Lightweight state management
- 🎨 **Tailwind CSS 4** + **DaisyUI**
- 🔥 **React Hot Toast** – Toast notifications
- 🔄 **React Router DOM 7**
- 📡 **Socket.IO Client**
- ✍️ **Lucide React** – Icon library
- 🌐 **Axios** – For API requests

### Backend
- 🟢 **Node.js**
- ⚙️ **Express.js**
- 🧠 **MongoDB + Mongoose**
- 📡 **Socket.IO Server**
- 🛡️ **JWT Authentication**
- 📤 **Cloudinary** (optional for image uploads)


---

## 🧪 Features

- 🔒 **User Authentication**
- 💬 **1-on-1 Real-time Messaging**
- 🖼️ **Image Support in Chat**
- 🌐 **Online Users Indicator**
- 📲 **Responsive Design**
- ⚡ **Socket.IO** for instant updates
